Difference between HTML tags and HTML attributes.

HTML Tag HTML Attributes
1. Tags are used to hold the HTML element. 1. Attributes are used to describe the characteristic of an HTML element.
2. Tag starts with '<' and ends with '>'. 2. Attributes are found only in the starting tag.
3. Tags are almost like keywords where every single tag has unique meaning. 3. Attributes specify various additional properties to the existing HTML element.
